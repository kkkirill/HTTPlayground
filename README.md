## HTTPlay~~ground~~

#### _Custom HTTP Handler_

##### Requirments:
* python 3.7.x
* Jinja2 2.10.1

##### Installation dependencies:

    pip install -r requirements.txt
    
##### Usage:
* Run server:
    
      python run_server.py <URL> <PORT>
    
* Check "URL:PORT/form" specified in _HTTPlayground/settings.py_ or passed by you as arguments
